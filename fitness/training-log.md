# Training log

Newest entries at top. Add a new entry after each session — takes ~2 minutes.
Running sessions from Runna just need a quick completion note + how you felt.

---

## Log entry template

Copy this, fill in, paste at the top:

```
## YYYY-MM-DD — [Session type: Upper / Lower / Full Body / Easy Run / Tempo / Long Run / Rest]
**Mood/Energy:** X/10
**Sleep:** Xh (quality: good / ok / poor)
**Soreness:** [none / mild legs / mild upper / heavy]

**Session:**
- Exercise: sets×reps @ weight (RPE X)

**Notes:** [anything worth flagging — tight spots, felt strong, cut short, etc.]
```
